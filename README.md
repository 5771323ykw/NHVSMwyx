# 前言

欢迎来到基于SSM的高校报修管理系统项目！本项目旨在为高校提供一个便捷、高效的报修管理系统，使报修流程更加规范化、透明化。通过本项目，我们希望为广大师生提供一个良好的报修体验。以下是本项目的详细介绍。

# 内容介绍

本项目后端采用Java语言，结合Spring、Spring MVC和MyBatis框架，前端使用JS、Vue和CSS3技术。系统主要包括以下几个模块：用户管理、报修单管理、维修人员管理、设备管理等。用户可以通过系统快速提交报修单，维修人员可以实时接收到报修信息并进行处理，管理员可以查看报修进度和维修人员工作情况。

# 技术介绍

- 语言：Java
- 使用框架：Spring、Spring MVC、MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
